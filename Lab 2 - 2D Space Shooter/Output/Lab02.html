Project Detail:
Basic 2D shooter game, created using Unity and following the Walker Boys Studio lessons.
Followed the tutorials with slight alterations - Used C# instead of JavaScript, so I had to make some changes. The particle system was also changed.
I also did all the "On Your Own" exercises and the Extra Credit exercise.


Tools:
This lab was made using Unity, with C# and Visual Studio 2012 (On my own) as an editor. I used Paint.net to edit the pick-up textures.


Final Thoughts:
This project was a bit more complicated, but it's pretty impressing what you can do using Unity and just a little bit of code.
This time, I strayed a bit more on the final "On my own" phase, creating some things that where not on the exercises. I also
tried my hand creating 3D models, but didn't get any good results with it, so went with the basic forms.
Unfortunately, I had a lot less free time for this lesson, so there are a few ideas I left unimplemented, but I'll probably come
back to it in the future.
